1. f-utils-absolute-all

应用场景：

不清楚父元素的宽高，却想与父元素比肩。只要父元素上有样式position-relative,就能占满父元素的空间

2. f-utils-fill

应用场景：

a. 没有基础宽度,占剩余所有空间，并且超出隐藏，不出现滚动条

b. 父元素是流式布局，可以让该子元素占剩余所有空间。如果父元素是横向布局，占剩余所有宽度。如果父元素是纵向布局，占剩余所有高度

c. 如果兄弟元素也有这个样式，平分剩余所有空间。

3. f-utils-fill-auto

应用场景：

a. 没有基础宽度,占剩余所有空间，并且超出横向隐藏，纵向出现滚动条

b. 其他同f-utils-fill。

4. f-utils-flex-column

应用场景：

内部流式布局，子元素要纵向布局。配合.f-utils-fill,可以让某个子元素占剩余所有空间。

5. f-utils-fill-flex-column

应用场景：

占父元素的剩余所有空间，并且内部流式布局，子元素是纵向布局。相当于f-utils-fill 和f-utils-flex-column

6. f-utils-flex-row

应用场景：

内部流式布局，子元素是横向布局，不能自动折下一行。常用于处理只有一行，并且内部子元素可以继承到父元素的高度

7. f-utils-fill-flex-row

应用场景：

占父元素的剩余所有空间。内部流式布局，子元素是横向布局，并且超出隐藏。

8. f-utils-flex-row-wrap

应用场景：

内部流式布局，子元素是横向布局，并且超出宽度自动折下一行。

9. f-utils-fill-flex-row-wrap

应用场景：

占父元素的剩余所有空间。内部流式布局，子元素是横向布局，并且超出宽度自动折下一行。
