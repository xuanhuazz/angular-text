1. 输入控件的基本结构

应用场景：

a. 输入控件有基本的结构，用来控制布局方向、禁用、只读央视，并控制第三方引入控件的样式。注意form-control的位置。

b. 默认是有最大宽度限制，如果是要求某个输入控件解除宽度限制，增加farris-group-auto。

2. Checkbox布局

应用场景：

a. 默认标签是在前面

b. 只对齐标签在后面。注意label标签的for属性。如果前面标签中没有文字，把for属性去掉。

c. 前标签不占位置。需要增加 f-checkradio-no-label

3. 自行构建checkbox组

应用场景：

a. 默认纵向布局

b. 横向布局，增加farris-checkradio-hor。

4. 单选多选在表格

应用场景：

a. 增加classf-checkradio-single

表格启用checkbox结构
表格启用radio结构
